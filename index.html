Definition



HTML stands for Hyper Text Markup Language. HTML is the standard markup language for creating Web pages. HTML describes the structure of a Web page.>>>

History



The first version of HTML was written by Tim Berners-Lee in 1993. Since then, there have been many different versions of HTML. Nowadays most widely used version of HTML is the 5th one, launched in 2014.

Syntax



The main syntax of html coding language is based on opening and closing angle brackets in between of these two command brackets, the instruction(s) will be held.

*IMAGES*









Introduce an image by using "img src='image.name'"

{Other functions}:

  • Control the image's *size* by using height="x" and width="y"

  • Adding *alt text* is considered a good practice

  • A *title* is needed too

  • An *image* can be a hyperlink as well

*AUDIO*









Introduce an audio by using "audio controls src='audio.name' ... /audio"

{Other functions}:

  • "autoplay" attribute will make the audio play automatically

  • "muted" attribute will the mute the audio mute by default

  • "loop" attribute will repeat the audio non-stop
